NAME
  capi20proxy (CAPI Services over TCP/IP)
SYNOPSIS
  n/a
DESCRIPTION
  This project is meant to provide ISDN CAPI 2.0 services (like fax) over 
  a UDP/IP based network. Clients (Win32) can use a CAPI interface (capi2023.dll)
  without having an ISDN card installed. These services are provided by a
  server (Win32/Linux/Unix) whith an ISDN card installed
